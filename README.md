# BHG Edge Demo

A comprehensive demo application for Blue Horizons Group's Professional Services OS platform.

## Overview

BHG Edge is a modern, full-featured demo showcasing Sales Performance Management (SPM) consulting capabilities.

## Features

- Dashboard with 6P Framework
- Sales: CPQ, Pipeline, Forecasting  
- Marketing: Campaigns, Social Media, Content Hub
- Delivery: Projects, Scoping Tool, Assets, AI Workbench
- Operations: IT Systems, Finance, Team Directory

## Tech Stack

- Next.js 15 with App Router
- TypeScript
- Tailwind CSS
- Radix UI Icons

## Getting Started

```bash
pnpm install
pnpm dev
```

Access code: **bhg-demo-2025**

## Deploy

Optimized for Vercel deployment.
